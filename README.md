# qdrant-hybrid-search-demo

This repo is a Barebones demo of Qdrant hybrid search running on Qdrant Hybrid Cloud.

### Assumptions
1. Qdrant v1.12.4
2. Python v3.11.7
3. Kubernetes info: \n
  a. Client Version: v1.24.11-dispatcher
  b. Kustomize Version: v4.5.4
  c. Server Version: v1.31.0
4. Minikube v1.34.0
